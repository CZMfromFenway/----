
游戏详解

0 前言

《皇室导弹战争：融合版》是对Cubehamster等人在1.8时代的作品《导弹战争》的复刻。该复刻版在原作的基础上加入了新设置、新导弹、融合机制和皇室模式。
游戏详解共分为玩法概要、规则详解、道具介绍和指令集等部分。
初涉此小游戏，仅需阅读“玩法概要”和“道具介绍”即可。

1 玩法概要

使用/trigger menu调出菜单，可以从此面板中开始游戏。
游戏开始后，双方会定期得到随机道具，包括防御性道具和用于部署导弹的怪物蛋。
持怪物蛋在平台前端的地面上右键即可在前下方部署相应导弹，使用防御性道具可以摧毁或阻拦敌方导弹。
用导弹破坏对方平台并摧毁对方地狱门即可获胜。

此改版新增多种融合导弹和融合道具，某些导弹刷怪蛋两两组合可合成新的导弹，防御性道具两两组合可以产生新的道具。
（具体方法见“道具介绍”章节）

皇室模式提供全新的选卡系统，玩家可以选择8种不同的道具出战，消耗费用来使用道具。皇室模式通过依次点击菜单->游戏设置->游戏模式->皇室来选中。
在此模式下，开始游戏并重置场地后，玩家通过把背包中的道具放至快捷栏中选卡，选卡结束后使用/trigger ready，当所有玩家都准备就绪时游戏开始。快捷栏中的道具数量对应费用。此模式下无法融合道具。

3 规则详解

3.1 组队规则

随机组队：在游戏开始时，玩家被随机分配到绿队和黄队，当玩家人数为奇数时，双方人数不平衡，人数少的队伍回费或补给会加速。
手动组队：游戏开始前玩家通过组队界面手动组队，无人组队时游戏无法开始，游戏开始时未组队的玩家将观战。

3.2 游戏规则

3.2.1 多元规则

游戏模式
普通：无特性，所有规则均可调节。
分钟：补给时间固定为60s，即每分钟得到所有基础导弹和防御性道具，此模式下不可禁用防御性道具。
皇室：在此模式下补给时间和箭矢数量将不可用，不可禁用防御性道具，玩家游戏开始前自选卡组并通过在游戏中消耗费用部署道具以作战。

行动增益
普通移速：不会获得速度或跳跃提升效果。
神行太保：获得速度I和跳跃提升II。
齐天大圣：获得速度IV和跳跃提升VI。

平台材料
更改双方的平台材料，五个可选项按抗爆炸性排列为：
玻璃、羊毛、带釉陶瓦、混凝土、陶瓦

3.2.2 二元规则

死亡掉落：禁用则开启死亡不掉落，但玩家每次重生后都会获得弓。
破坏方块：启用则可以手动拆除敌方导弹和盾等（启用则游戏时为生存模式，否则为冒险模式）。
防御道具：启用则补给盾、火球、箭矢三种基础防御性道具。
火球防御：禁用则火球会在地狱门前爆炸，启用则火球无法摧毁地狱门。
部署位置：“平台”意味着我方导弹仅可在我方平台附近部署，其他位置部署只会消耗怪物蛋；“全局”意味着双方导弹可以在任意位置部署。
道具上限：双方均会定期得到道具补给，启用则补给时不会获得已有的道具。
反向消退：启用则消退模式中平台从地狱门侧开始消失。

3.2.3 数值规则

补给时间：双方所有玩家都会定期得到补给，此选项设置补给的周期。皇室模式的回费间隔为此周期的一半。
箭矢数量：每次补给时有概率获得箭矢，此选项设置每次补给箭矢的数量（若物品上限开启，玩家仅在耗尽箭矢后才能被补给箭矢）。
消退时间：启用后，游戏中双方的玩家以此值为周期逐层消失，同时适时启动双倍圣水模式。
平台距离：双方平台之间的空间长度（实际上，此选项设置的是平台距双方中点的距离）。
平台厚度：双方平台均由厚度相等的三层组成（每层颜色相同），此选项设置单层的厚度。

4 指令集

4.1 功能触发指令

使用指令
/trigger 功能名

不同的功能对应如下表

功能名	实现功能
menu	调出菜单
start	申请开始游戏
stop	在游戏进行时强制停止
settings查看并修改游戏设置
team	查看并修改组队设置
help	获取帮助手册
ready	完成选卡

4.2 手动组队指令

手动加入绿队
/trigger join set 1
手动加入黄队
/trigger join set 2
